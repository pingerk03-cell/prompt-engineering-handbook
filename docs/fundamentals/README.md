## Core idea

A prompt is not just a sentence.

A professional prompt is a structured instruction system.

It tells the AI:

- what goal to achieve;
- what role to take;
- what context to use;
- what task to complete;
- what format to follow;
- what constraints to respect;
- how success should be judged.
