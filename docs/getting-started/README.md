---

## Core Idea

Prompt Engineering is not about memorizing magic phrases.

It is about designing clear communication between a human and an AI system.

A good prompt helps the model understand:

- the goal;
- the role;
- the context;
- the task;
- the output format;
- the constraints;
- the quality criteria.

---

## Practice Mindset

Do not try to write perfect prompts from the beginning.

Start with a simple version, test it, observe the result, and improve it.

Prompt Engineering is an iterative process.

Small improvements matter.
